# My Finance — mobile app for personal finance management

A simple mobile application for tracking income, expenses, and transfers between accounts. Developed with 1С:EDT.

## 🛠️ Tech Stack

- 1С:Enterprise 8.3 platform
- 1С:Enterprise Development Tools (EDT)
- Git

## 🚀 How to Run

1. Clone the repository
2. Open the project in EDT
3. Set up a mobile application launch configuration

## 📋 Status

The project is under active development. Basic accounting logic and currency functionality are implemented.
